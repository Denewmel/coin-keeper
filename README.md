# CoinKeeper - Трекер расходов

## Описание
Простое приложение для учёта личных финансов. Помогает отслеживать доходы и расходы.

## Основные функции
- Добавление доходов и расходов
- Просмотр истории операций
- Отображение текущего баланса
- Фильтрация по категориям
- Получение курса валют через API

## Структура экранов
1. **Главный экран** - баланс, кнопки добавления, последние операции
2. **Экран транзакций** - полный список всех операций
3. **Экран добавления** - форма для новой операции

## Навигация
Главный → Все транзакции  
Главный → Добавить операцию  
Все транзакции → Главный (назад)  
Добавить операцию → Главный (после сохранения)

## Сценарии использования

### Сценарий 1: Добавление расхода
1. Открыть приложение → Главный экран
2. Нажать кнопку "РАСХОД"
3. На экране добавления:
   - Сумма: 1500
   - Категория: "Продукты"
   - Описание: "Магазин у дома"
4. Нажать "СОХРАНИТЬ"
5. Автоматический возврат на Главный экран
6. Видим обновлённый баланс и новую запись

### Сценарий 2: Просмотр истории
1. Открыть приложение → Главный экран
2. Нажать "ВСЕ ТРАНЗАКЦИИ"
3. Видим полный список операций
4. Можем фильтровать по "Только расходы"
5. Кнопка "Назад" возвращает на главный
